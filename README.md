# EreborHub — Premium Technology Solutions

EreborHub engineers secure infrastructure and identity platforms that empower modern businesses to scale with confidence and impact.

## 🚀 Overview

EreborHub is a high-end digital agency and technology partner specializing in building resilient, scalable, and premium software solutions. From robust backend infrastructures to sophisticated identity verification systems, we transform technical complexity into elegant business tools.

## 🛠 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **ORM**: [Prisma](https://www.prisma.io/)
- **Internationalization**: Custom i18n routing with support for English and French
- **Icons**: [Lucide React](https://lucide.dev/)

## 🌐 Features

- **Multilingual Support**: Fully localized content and SEO (i18n)
- **High-Performance UI**: Smooth animations and premium "Deep Tech" aesthetic
- **SEO Optimized**: Dynamic metadata, structured sitemap, and robots.txt configuration
- **Responsive Design**: Optimized for all devices, from mobile to ultra-wide displays
- **Enterprise-Grade Data**: Structured data models for projects, services, and testimonials

## 🏗 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app/[locale]`: Multilingual page routing
- `src/components`: Reusable UI components (Hero, CTA, Projects, etc.)
- `src/lib/data`: Centralized data for services and portfolio
- `src/lib/dictionaries`: Translation files (JSON)
- `public`: Static assets (Optimized images, Open Graph vignettes)
